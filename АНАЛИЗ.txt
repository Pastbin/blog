Анализ приложения:

- Приложение разработано на стеке технологий:
• Laravel
• Vue.js
• Bootstrap
• PHP
• SQlite
• JavaScript
• Axios
• Git
• SCSS
• Vite

- Созданы все необходимые контроллеры, модели, роуты и миграции, а так же визуальное представление.

- Для быстрого прототипирования были задействованы такие инструменты как:
• php artisan make
• php artisan tinker
• composer
• SQlite
• Laravel
• Bootstrap.

- Компоненты разбиты на модули для удобства масштабирования и переиспользования.

- Был использован frontend framework Vue для модульности приложения и его компонентов.
Компонентный подход позволяет разработчикам быстро создавать новые компоненты и переиспользовать существующие.

- В ходе разработки были использованы следующие команды:
• php artisan make позволяла быстро генерировать все необходимые контроллеры, модели, миграции.
• php artisan tinker позволяла просматривать БД.
• php artisan route:list выводила список всех роутов.
• php artisan migrate помогала создавать и изменять БД.
